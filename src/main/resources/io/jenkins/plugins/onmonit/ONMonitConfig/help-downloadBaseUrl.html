When set, the node_exporter and otel-contrib binaries are downloaded from this URL by appending / and one of the filenames below. OS can be either linux or darwin. Arch can be either amd64 or arm64. Windows binaries are only supported for amd64.
  1. node_exporter-1.5.0.os-arch
  2. windows_exporter-0.22.0-amd64.exe
  3. otelcol-contrib_0.70.0_os_arch
  4. otelcol-contrib_0.70.0_win_amd64.exe
