When set, the node_exporter and otel-contrib binaries are downloaded from this URL by appending / and one of the filenames below. OS can be either linux or darwin. Arch can be either amd64 or arm64. Windows binaries are only supported for amd64.
  • node_exporter-1.5.0.os-arch
  • windows_exporter-0.22.0-amd64.exe
  • otelcol-contrib_0.70.0_os_arch
  • otelcol-contrib_0.70.0_win_amd64.exe
E.g. when https://example.com/build-repository and the build agent is Linux on amd64, then using the onMonit step will download https://example.com/build-repository/node_exporter-1.5.0.linux-amd64 and https://example.com/build-repository/otelcol-contrib_0.70.0_linux_amd64 if not already present on the PATH.
